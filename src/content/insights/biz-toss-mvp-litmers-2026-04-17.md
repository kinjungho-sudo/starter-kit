# 토스 MVP 전략의 교훈 - 지금도 프리토타이핑이 통할까?

## 핵심 요약
토스는 2014년 랜딩페이지 + 광고비 1만원으로 시장 수요를 검증했다 (프리토타이핑).
지금은 광고비 급등 + 사용자 기대치 상승 + 경쟁 심화로 동일한 방식이 통하지 않는다.
초기 창업자의 해법: 핵심역량에 집중하고 나머지는 외부 전문가(프리랜서/에이전시)로 보완.

## 주요 내용

### 토스의 초기 전략 (2014)
- "10초 송금" 메시지 랜딩페이지 제작
- 페이스북 광고 **1만원 투자 → 6,000명 노출 → 24명 클릭**으로 수요 검증
- 앱 없이 대표가 직접 인터넷뱅킹으로 송금 → 3일 만에 2,000명 전화번호 확보
- 핵심 교훈: 제품 없이도 시장성 먼저 검증 가능

### 프리토타이핑이란?
- 제품 개발 **전에** 최소 리소스로 시장 반응을 검증하는 방법
- 프로토타입보다 더 빠르고 저렴
- IBM 사례: 음성인식 기술 개발 전 사람이 직접 입력하는 방식으로 수요 먼저 확인

### 지금은 왜 안 통하나?
| 2014년 | 2024년 |
|--------|--------|
| 페이스북 CPC 매우 저렴 | 광고비 수십 배 상승 |
| MVP = 기본 기능으로 충분 | MVP도 높은 완성도 요구 |
| 경쟁 플레이어 적음 | 카카오페이·네이버페이 등 대형 플레이어 존재 |
| 아이디어만으로 차별화 가능 | AI·데이터 등 기술 전문성 필수 |

### 지금의 해법
- **핵심역량 집중**: 창업자가 가장 잘하는 1가지에 자원 몰기
- **외부 전문가 활용**: 마케팅·개발·디자인은 프리랜서/에이전시로
- **프로젝트 기반 협업**: 장기 고용 대신 필요할 때만 활용 → 비용 리스크 감소
- McKinsey: 미국 기업 **70%**가 프로젝트 단위 외부 전문가 활용으로 전환 중

## 인사이트

### 코마인드웍스 사업 연결
- **직접 적용 가능**: 코마인드웍스가 제공하는 AI 에이전트·자동화 서비스가 곧 "핵심역량 외의 영역을 외부에서 보완"하는 솔루션
- **랜딩페이지 검증 프레임**: 새 서비스 출시 전 랜딩페이지로 수요 먼저 확인 → 지금도 유효한 방법론 (단, 광고비 효율 계산 필수)
- **콘텐츠 소재**: "토스가 했던 방식 vs 지금 해야 할 방식" → 유튜브/블로그 비교 콘텐츠로 활용 가능
