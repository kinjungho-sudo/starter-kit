# 마이크: 실패 없는 SaaS 창업 10단계 플레이북 - 월 2억 7천만원 부트스트랩

> 호주 연쇄 창업가 마이크. 5개 SaaS 부트스트랩 운영, 월 2억 7천만원. 런칭한 모든 앱 실패 없음. 비결: 혁신 대신 지루하지만 검증된 아이디어 + 10단계 반복 플레이북.
> 채널: 동기부여학과

---

## 핵심 인사이트

**혁신 추구 = 실패 지름길. 이미 검증된 시장에서 더 나은 UX로 승부하면 실패할 수 없다.**

## 마이크의 5개 SaaS

| 서비스 | 카테고리 | 설명 |
|--------|---------|------|
| Curated.io | 소셜 미디어 | 웹사이트/이벤트에 여러 소셜 콘텐츠 통합 |
| Frill.co | 고객 피드백 | 피드백 수집 → 로드맵 반영 → 기능 공지 |

**합산 월 매출: 2억 7천만원 이상**

## 운영 철학

- **부트스트랩**: 외부 투자 없이 창업자끼리 수익 나눔
- **최소 팀**: 큰 엑시트가 아닌 큰 연봉이 목표
- **5년 목표**: 월 13억원, 여전히 최소 팀 유지
- **광고비 최소화**: 콘텐츠 마케팅으로 유기적 성장

## 아이디어 선별 기준

### 하는 것 ✅
**"나쁜 UX에도 불구하고 잘 팔리는 제품"**을 찾아 더 좋게 만든다

```
Frill.co 사례:
기존 피드백 툴 → 복잡하고 UX 나쁨 → 그럼에도 많은 사용자 보유
→ "더 좋은 UX로 만들면 당연히 더 성공한다"는 논리
```

### 하지 않는 것 ❌
**AI 의존 비즈니스** 절대 안 함

이유:
- OpenAI 정책 변경/가격 인상 → 하루아침에 붕괴 가능
- 구글/MS가 무료로 유사 서비스 출시 → 경쟁 불가

### 관심 분야 🎯
**문서와 도구** - AI 시대에 잘 정리된 문서가 제품 성공의 핵심이 됨
(AI가 제품 추천/설명 시 정확한 문서 학습 → 우위 확보)
단, AI는 보조 도구로만 활용. 핵심 기능은 본질적 가치에 집중.

## 10단계 플레이북

### Phase 1: 런칭 준비
**1단계: 검증된 아이디어 선택**
- 새로운 아이디어 ×, 이미 시장이 검증한 카테고리 선택

**2단계: MVP 정의**
- 경쟁사 분석 → 고객이 가장 원하는 핵심 기능만
- 완벽한 제품 금지, 핵심만 빠르게 출시

### Phase 2: 초기 자금 확보
**3단계: 평생 할인가 제공**
- 8~13만원 일회성 결제로 평생 이용 상품 출시
- 목적: 초기 자금 확보 + 실제 사용자 확보

**4단계: 무료 계정 절대 금지**
- "돈을 내야 실제로 씀" - 유료 사용자만이 진짜 피드백을 줌
- 가능한 많은 피드백 수집이 목표

**5단계: 직접 발로 뛰어 평생 할인 판매**
- 레딧 그룹 활동, 비공개 페이스북 그룹 참여, X 소통
- Frill.co는 이 단계에서 약 4천만원 모금

### Phase 3: 지속 성장 기반 구축
**6단계: 콘텐츠 시작**
- 랜딩페이지 + 블로그 + 경쟁사 비교 페이지
- 구글 + 챗GPT 인덱싱 목표
- "콘텐츠를 만들기에 너무 이른 때는 없다"
- 평생 할인 판매 수익 → 콘텐츠 제작에 재투자

**7단계: 앱수모/마켓플레이스 런칭**
- 더 넓은 사용자 + 추가 자본 확보
- 목표: 평생 할인 종료 시 13억원 확보 → 1~2년 콘텐츠 투자 재원

### Phase 4: 전환점
**8단계: 마지막 평생 할인 진행**
- 가격 소폭 인상 + "정말 마지막 기회" 어필 (FOMO 활용)
- 이메일 리스트 발송 후 종료

**9단계: 리뷰 확보**
- 1억원 자금으로 기존 평생 구독자에게 리뷰 요청
- Trustpilot, G2 등 - 평생 구독자는 성공을 원하므로 적극 협조

**10단계: 레딧 활동**
- 경쟁사 찾는 질문에 정직하고 진정성 있게 답변
- 이 단계에서 자금은 바닥나지만 월정기 매출이 시작되어야 함
