# 생각에 관한 생각: 시스템1·2, 휴리스틱과 편향 - 10분 요약

> 대니얼 카너먼 《Thinking, Fast and Slow》 핵심 요약. 1secret 뉴스레터.
> **상위 1%와 우리의 차이는 생각하는 방법에서 시작된다.**

---

## 핵심 프레임: 두 시스템

| | 시스템 1 | 시스템 2 |
|---|---|---|
| 속도 | 빠름 | 느림 |
| 특성 | 자동, 직관적, 무의식 | 의식적, 논리적, 노력 필요 |
| 비유 | 오토파일럿 | 숙련된 조종사 |
| 장점 | 효율적, 빠른 판단 | 복잡한 문제 해결 |
| 단점 | 체계적 오류 범함 | 느리고 에너지 소모 큼 |

---

## 제1부: 두 시스템

### 1장: 두 시스템
- 시스템 1: 2+2=? 처럼 즉각적으로 답이 나오는 판단
- 시스템 2: 17×24=? 처럼 의식적 계산이 필요한 판단
- 대부분의 일상 결정은 시스템 1이 처리 → 효율적이지만 오류 위험

### 2장: 주의와 노력
- 시스템 2는 **제한된 자원** - 주의력과 노력이 소모됨
- 인지적 과부하(멀티태스킹, 스트레스) 상태에서 시스템 2가 약화
- → 시스템 1의 오류를 검증하지 못하게 됨

### 3장: 게으른 제어
- 시스템 2는 에너지를 아끼려 시스템 1을 그냥 통과시킴

**박쥐와 공 문제:**
```
박쥐와 공의 합계 = 1.10달러
박쥐가 공보다 1달러 비싸다
공의 가격은?

직관(시스템 1) → 10센트 (틀림)
계산(시스템 2) → 5센트 (정답)
```
→ 대부분의 사람이 시스템 2를 작동시키지 않고 오답을 낸다

---

## 제2부: 휴리스틱과 편향

### 4장: 연관 기계 - 상관관계 ≠ 인과관계

> **아이스크림 판매량 ↑ → 익사 사고 ↑**
> 실제 원인: 여름 더위 (공통 원인)

시스템 1은 두 사건이 함께 발생하면 자동으로 인과관계를 추론한다.
→ 창업/투자 판단에서 이 편향을 인식해야 함

### 5장: 인지적 편안함 - 단순 노출 효과

- 익숙하고 이해하기 쉬운 정보를 더 **신뢰**하고 더 **좋아**함
- 반복 노출만으로도 선호도 증가 (광고·브랜딩의 원리)
- 이해하기 쉬운 이름·설명·디자인이 더 좋은 평가를 받음

> **적용**: 랜딩페이지·콘텐츠는 인지적 편안함을 극대화해야 전환율이 높아진다

### 7장: 대체 휴리스틱 - 어려운 질문을 쉬운 질문으로 바꾼다

원래 질문 → 대체 질문으로 무의식 교체

| 원래 질문 | 대체 질문 |
|---------|---------|
| 이 후보가 대통령으로 얼마나 적합한가? | 이 사람이 얼마나 인상적인가? |
| 이 투자가 5년 뒤 얼마나 성장할까? | 이 사람이 얼마나 자신 있어 보이는가? |
| 이 제품이 시장에서 살아남을까? | 이 아이디어가 얼마나 매력적으로 느껴지는가? |

→ **우리는 어려운 질문에 직접 답하지 않고 더 쉬운 질문에 답한 뒤 그것을 원래 답으로 착각한다**

---

## 핵심 교훈 3가지

1. **직관을 믿되 검증하라** - 시스템 1은 강력하지만 중요한 결정에선 시스템 2로 검증
2. **인지적 편안함을 설계하라** - 콘텐츠·제품·메시지를 쉽고 익숙하게 만들면 신뢰도 상승
3. **진짜 질문에 답하라** - 내가 지금 대체 휴리스틱으로 쉬운 질문에 답하고 있는 건 아닌지 점검

## 적용 포인트 (코마인드웍스)

- [ ] 고객이 랜딩페이지에서 **시스템 1**으로 "이게 뭔지" 3초 안에 이해하게 설계
- [ ] 투자·사업 결정 시 "인상적으로 느껴지는가?" 대신 "실제 지표가 검증됐는가?" 질문
- [ ] 콘텐츠에 **단순 노출 효과** 활용: 반복 접점 → 신뢰도 자동 상승
- [ ] "이 아이디어가 매력적이다 ≠ 이 시장이 검증됐다" 분리 사고 훈련
